\section{Why Hallucination Detection Requires Temporal Localization}
\label{sec:non_static}

Mechanistic detectors face a tension among coverage, localization,
and scalability. Fixed-position probes \citep{azaria2023,orgad2024} are
efficient but read only one layer-token slice, so they miss signals
that appear elsewhere in the activation tensor. Full-response
activation maps capture broader structure, but they are expensive and
usually collapse the response into one sequence-level score
\citep{barshalom2025}. Claim-level detectors provide coarser
localization, but scoring extracted claims in isolation ignores
boundary evidence from neighboring supported and hallucinated spans
\citep{snyder2024,he2024}. Long-form hallucination detection therefore
requires an inference unit that is local enough to assign span-level
scores, broad enough to include adjacent context, and cheap enough to
run across a long response.

\paragraph{Full-response activation maps do not scale to long-form localization.}
ACT-ViT shows that layer-token activation structure is predictive, but
full-response pooling is the wrong inference unit for long-form
localization. Appendix Figure~\ref{fig:perf_by_length} reports
per-token AUROC on LongFact as the input prefix length $T$ grows:
ACT-ViT with a short pooled token axis is fast but loses accuracy as
more generated tokens are compressed into the same representation.
Widening the pooled axis partially reduces this collapse at the cost
of inference speed, and Figure~\ref{fig:inference_time} shows the
runtime penalty when a full-response activation map is rescored at
each token. TPE instead scores local token windows and models
their temporal evolution, avoiding one pooled representation for the
whole response.

\paragraph{Why windows instead of isolated tokens?}
HaMI \citep{niu2025hami} shows that hallucination evidence can be
sparse: only a small part of a response may carry the decisive signal.
TPE shares this sparsity assumption, but changes the localization
unit from individual token representations to local activation windows.
The evidence is not confined to isolated token states; it forms a
local temporal pattern around the hallucinated span. Figure~\ref{fig:signal_propagation}
measures this directly on RAGTruth: a linear probe's ROC-AUC ramps
from chance at offset $-24$ to $0.66$ at the token immediately before
a hallucinated span, rises through the span ($0.71 \to 0.79$), and
decays slowly afterward. Unlike HaMI-style token selection, where
individual token representations are the MIL instances, TPE uses
local activation windows as instances because useful evidence appears
before, during, and after the labeled span.

\paragraph{Why encode activations as 2D images?}
Per-layer and per-token probes show that useful hidden-state signal is
not concentrated in one fixed position. Informative regions appear
across both the layer direction and the token direction, and the most
predictive regions change across LLMs, as shown by the heatmaps in
Appendix~\ref{app:heatmaps}. A single layer-token slice or one-axis
pooling can discard cross-axis patterns in this layer-token plane.
ACT-ViT \citep{barshalom2025actvit} motivates activation-image encoding by
treating the layer-token activation plane as a 2D image and showing
that visual encoders improve over fixed-position probes. TPE
keeps this activation-image view, uses Perception Encoder features
\citep{bolya2025perceptionencoder}, and changes the inference unit
from one pooled sequence image to a temporal sequence of local
activation images.

\paragraph{Weakly supervised temporal localization.} ACT-ViT shows
that the $(L,T)$ plane has spatial structure amenable to visual
encoders, but a single fixed-size image fits poorly when $T$ ranges
from a handful to thousands of tokens. We frame $A$ as a video over
the token axis and borrow weakly supervised temporal localization
from video understanding \citep{pilhyeonlee2021}. Each token is a
frame whose height is the layer axis and whose channels are the
hidden dimension; a window aggregates $W$ consecutive frames into a
local activation image. A claim or response label then provides weak
supervision over a temporal sequence of windows. We use top-$k$ MIL
over windows, not over isolated token states, so the model learns
which local activation windows explain a positive label without
window-level annotations.

\paragraph{Three-stage training.} Learning sequence-level uncertainty
and window-level localization jointly under weak supervision is a hard
optimization, so we decompose training into three stages. Stage~1
encoder pretraining learns uncertainty-sensitive activation patterns
from isolated labeled claims. Stage~2 top-$k$ window localization
fine-tunes the encoder with a temporal head \citep{albertgu2024},
using MIL to identify the few local windows that explain a positive
claim label. Stage~3 cross-claim refinement trains on whole passages
with neighboring correct and hallucinated claims, exposing the model
to mixed-content windows and boundary behavior that isolated-claim
training does not cover. We describe each stage in the next section.
